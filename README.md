# Desktop Cat

A lightweight cross-platform desktop pet built with Rust and Tauri.

This orange-and-white British Shorthair cat lives on your desktop, walks around the screen, sleeps, stretches, plays, and reacts to mouse interactions. Designed with low CPU and memory usage in mind, it runs smoothly on Windows and macOS.

## Features

- 🐱 Animated desktop cat
- 💤 Sleeping, walking, sitting, and idle behaviors
- 🎯 Mouse interaction and tracking
- 🔄 Smooth animation system
- ⚡ Low resource consumption
- 🖥️ Windows & macOS support
- 🎨 Custom character and animation support
